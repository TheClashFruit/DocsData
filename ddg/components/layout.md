Either a `HorizontalLayout` or a `VerticalLayout`.

## Example

```xml
<HorizontalLayout gap="8">
  <!-- ... -->
</HorizontalLayout>
```

```xml
<VerticalLayout>
  <!-- ... -->
</VerticalLayout>
```

## Attributes

| Attribute | Type         | Description                                 |
|-----------|--------------|---------------------------------------------|
| `position`| `(int; int)` | The position of the layout in the menu.     |
| `gap`     | `int`        | The gap between the elements in the layout. |

\* Required.
